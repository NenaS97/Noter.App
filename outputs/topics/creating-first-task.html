Create Your First Task

Learn the steps to add a new task and set a basic reminder in Noter.app.

Follow these steps to create a new to-do item. You will also learn how to set a simple reminder for it in Noter.app.

  1. Go to Tasks
    1. Find the "Tasks" section in the Sidebar.
      Click on "Tasks". This opens the task management area. It appears in the main Note Editor Pane.
  2. Start a New Task
    1. Look for the "Add Task" button. It is a "+" icon, placed at the top of the task list. (See the Interface Overview)
      Click "Add Task" or the "+" icon. A new, empty task field will show. A cursor will blink, ready for your task description.
  3. Describe Your Task
    1. Type what you need to do in the new task field.
      Be clear. Be concise.
  4. Set a Basic Reminder
    1. Find the "Reminder" icon. It looks like a bell icon, placed near your new task. Click on it.
      A simple reminder window will appear. Choose a date and time. Click "Save" to confirm.
Your first task is now created. If you set one, a basic reminder is also active. The task is in your task list. You will get a notification at the set time.
